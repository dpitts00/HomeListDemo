#  User Stories

## Menu Item Stories

- I want to add a menu item.
- I want to delete a menu item (swipe action or edit button).
- I want to edit the details of a menu item - name, ingredients, meal, and price tier.
- I want to be able to search for (filter) the items by a string query, in name and ingredientList properties.
- I want to be able to sort the menu items ascending/descending by meal, price tier, alphabetically by name, with sections displayed for each sort group.

### WIP Menu Item Stories

- I want to tap on a menu item to select or deselect the item for the current week.
- I want to be able to filter the menu items by meal or price tier. 

## Household Item Stories

- I want to add an item
- I want to delete an item (swipe action or edit button).
- I want to edit the details of an item - name, room, user.
- I want to be able to search for (filter) the items by string query, in name, room, and user properties.
- I want to be able to sort the menu items ascending/descending by room, user, alphabetically by name.

### WIP Household Item Stories

- I want to be tap on an item to select or deselect the item from the current list.
- I want to be able to filter the items by room or user.

## Restaurant Stories

- I want to add a restaurant
- I want to delete an restaurant (swipe action or edit button).
- I want to edit the details of an restaurant - name, menu items, price tier.
- I want to be able to search for (filter) the restaurants by string query, in name and menuItems properties.
- I want to be able to sort the restaurants ascending/descending by price tier, or alphabetically by name.

### WIP Restaurant Stories

- I want to be tap on an restaurant to select or deselect the restaurant from the current list.
- I want to be able to filter the restaurants by price tier.
